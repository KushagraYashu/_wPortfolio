Key Points

At Alphadynamic, I was appointed as a Python Content Creator and was responsible for developing python language related educational products such as videos explaning code of a python language based game. Later, I was given the responsibility for the AI related projects. The AI projects were designed using Nyckel and MIT App Inventor tool. All of the work was targeted towards children. They are used to teach coding and AI to children.

Python related tasks:
Firstly, I was assigned to develop 2 simple python based games (balloon flight and space shooter).

I also worked on the videographic representation of these games. I developed the PPTs, scripts, videos, and finalised the whole project according to the need.

The games were simplistic yet interesting enough to occupy a child. The videos entertained and taught children about the game and its development.

I also recorded the Python and VS Code installation and configuration videos for the whole experience and support.

AI tools related tasks:
I was assigned to develop 4 patterns (rectangle, circle, infinity, and helix) based on the standard measurements and deploy them on the web (Vyorius Simulator Website) and Windows systems (.exe builds).

I implemented key logic for the scoring system, the scoring is very tight and based on the regulations. The logic is to find the number of times the player deviated from the correct path and the amount of deviation, the score is then calculated based on that information.

I also set up the environment for the Test Sim. The environment includes a city background with some characters overlooking the test.
