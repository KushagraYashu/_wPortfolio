1-Bit Jam 2023
Created Game: Passage
A Week-long jam to create a game using the theme "Light & Dark"
Key Points

The 1-Bit Jam, hosted on itch.io, tasked participants with creating a game in one week based on the theme "Light & Dark". This theme was designed to inspire developers to incorporate the concepts of light and darkness in their game designs, exploring how these elements can influence gameplay and aesthetics.

For the game "Passage", the theme was embraced by creating puzzle levels that utilize contrasting colors. Each level offers a unique environment with specific challenges that align with the theme. This approach not only enriches the gameplay experience but also ensures that the theme of "Light & Dark" is consistently represented throughout the game.

"Passage" is a color-based puzzle game where players navigate through various levels by solving puzzles. The primary gameplay mechanic involves pathfinding and exploration, with the objective being to successfully complete all levels. Each level is distinct in both color scheme and difficulty, providing a diverse range of challenges for the player.

The core mechanics of the game involve moving through colored panels while avoiding black or empty panels, which result in a loss of life. Additionally, the game features "switch" buttons that alter the puzzles, creating new paths and adding complexity to the solutions. This dynamic element keeps the gameplay engaging and challenging.

The game "Passage" is publicly accessible and can be downloaded from its dedicated itch.io project page. This platform allows players to experience the game firsthand and provides a space for feedback and community interaction.
