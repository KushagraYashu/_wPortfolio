Brackeys Game Jam 2023.2
Created Game: Dark World
A Week-long jam to create a game using the theme "Diving Deeper"
Key Points

The Brackeys Game Jam, hosted on itch.io, challenged participants to create a game within one week based on the theme "Diving Deeper". This theme encouraged developers to explore the concept of depth, whether literal or metaphorical, in their game designs.

For the game "Dark World", the theme was applied by designing different levels to represent various depths of the earth. Each level presents a unique environment and challenges that align with the concept of delving deeper into the ground. This setting creates an immersive experience, encouraging players to explore and conquer the unknown depths.

"Dark World" is a first-person game where players navigate through subterranean levels, each populated with demons. The core gameplay revolves around combat and exploration, with the primary objective being to defeat these demonic entities to save the protagonist's brother. This narrative provides a strong motivational driver for the player, adding emotional depth to the gameplay.
